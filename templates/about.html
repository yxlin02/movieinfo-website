Use of Kaggle

We searched the keywords “movie recommendations” on Kaggle and found this silver-rated (218 up-votes) dataset called Movie Industry: Four decades of movies collected by Daniel Grijalva, a student at Universidad de Sonora, through scraping the HTML from IMDb with a Python script. This dataset is also used in a GitHub Project created by Daniel, it also has 84.9k views and 16k downloads. It has a CC0 license and the data is in a .csv file.

Dataset Insight

We are using the second version of this dataset which is updated on July 23, 2021. We have downloaded a local copy of the dataset, as well as the GitHub repo with the scarping script used on January 23, 2022. The dataset contains movie data from 1980 to 2020 worldwide. In total, there are 7668 movies and each of them has the following attributes: name, rating, genre, year, released data, score, votes, director, writer, and movie stars participated.
